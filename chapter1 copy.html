Lecture 1 (Chapter 1)

Properties of numbers that all operations must obey:

  • (P1) If a, b, and c are any numbers, then (associative property):

    a + (b + c) = (a + b) + c

  • (P2) If a is any number, then (additive identity):

    a + 0 = 0 + a = a

  • (P3) For every number a, there is a number -a such that (additive inverse):

    a + (-a) = -a + a = 0

    Example: Proof: a + x = a when x = 0

    (P1) -a + a + x = a + (-a)
    (P1) (-a + a) + x = 0
    0 + x = 0
    (P2) 0 + x = 0, x = 0

    Note: This can be used to solve any equation.
  • (P4) If a and b are any numbers, then (commutative property):

    a + b = b + a

These are 4 properties for addition:

  • (P5) If a, b, and c are any numbers, then (associative property):

    a · (b · c) = (a · b) · c

  • (P6) If a is any number, then (multiplicative identity):

    a · 1 = 1 · a = a

  • (P7) For every number a ≠ 0, there is a number a-1 such that (multiplicative inverse):

    a · a-1 = a-1 · a = 1

    Example: If a · b = a · c, prove that b = c (a ≠ 0)

    (P7) a-1 · (a · b) = a-1 · (a · c)
    (P5) (a-1 · a) · b = (a-1 · a) · c
    1 · b = 1 · c
    (P6) b = c

    Example: If a · b = 0, prove that a = 0 or b = 0

    Assume a ≠ 0
    Proof: a-1 · (a · b) = 0 · a-1 will be proved next
    Hence, a-1 · a · b = 0
    1 · b = 0
    b = 0

    Note: This is used in solving the factorization of quadratic equations.
  • (P8) If a and b are any numbers, then (commutative property):

    a · b = b · a

  • (P9) If a, b, and c are any numbers, then (distributive law):

    a · (b + c) = a · b + a · c

    Example: Prove: if a - b = -a when a = b

    a - b + b = b - a + b
    a = b + b - a
    a + a = b + b - a + a
    a · 1 + a · 1 = b · 1 + b · 1
    a · (1 + 1) = b · (1 + 1)
    a · 2 · 2-1 = b · 2 · 2-1
    a · 1 = b · 1
    a = b

    Example: Prove a · 0 = 0

    a · 0 + a · 0 = a · (0 + 0)
    a · 0 + a · 0 = a · 0
    a · 0 + a · 0 - a · 0 = a · 0 - a · 0
    a · 0 = 0

    Example: Prove (-a) · b = -(a · b)

    (-a) · b + a · b = b · ((-a) + a)
    = 0 · b
    = 0
    (-a) · b + a · b - a · b = 0 - (a · b)
    (-a) · b = -(a · b)

    Example: Prove (-a) · (-b) = (a · b)

    (-a) · (-b) + (-(a · b)) = 0
    -(a · b) = (-(-a) · b) = (-a) · (-b) + (-a) · b
    = (-a) · ((-b) + b)
    = (-a) · 0
    = 0
    (-a) · (-b) + (-(a · b) + a · b = 0 + (a · b)
    (-a) · (-b) = (a · b)

    Example: Prove that (x - 1) · (x - 2) = 0

    x2 - 3x + 2 = 0

    (x - 1)(x - 2) = x(x - 2) + (-1)(x - 2)
    = x · x + (-2) · x + (-1) · x + (-1) · (-2)
    = x2 + x(-2 + -1) + 2
    = x2 + (-3)x + 2
    = x2 - 3x + 2

    Note: This strategy is used in the multiplication of any numbers.

Lecture 3

  • (P10) For every number a, only one of the following holds (Trichotomy Law):
    1. a = 0
    2. a is in P
    3. -a is in P
  • (P11) If a and b are in P, then (closure under addition):

    a + b is in P

  • (P12) If a and b are in P, then (closure under multiplication):

    a · b is in P

Definition: a > b ↔ a - b > 0

Example: If a < b and b < c, prove a < c

b - a > 0
c - b > 0
(b - a) + (c - b) > 0
c - a > 0
c > a

Definition: Absolute Value

|a| = a,   if a ≥ 0 -a,   if a < 0

Theorem 1: |a + b| ≤ |a| + |b| (Triangle Inequality)

Proof:
  • i) a > 0, b > 0

    a + b = a + b

  • ii) a < 0, b < 0

    -(a + b) = (-a) + (-b)

  • iii) a > 0, b < 0
    • Assuming (a + b) is positive

      a + b < a - b
      b < -b

    • Assuming (a + b) is negative

      -(a + b) = a - b
      -a - b = a - b
      -a < a

Note: Another proof for the same fact can be used with the definition |a| = √(a2)
