Spivak's Calculus - Chapter 1 Explanation

Chapter Overview

This chapter introduces fundamental properties of numbers and operations, laying the groundwork for more advanced calculus concepts.

Lecture 1: Properties of Numbers

Properties of numbers that all operations must obey:

(P1) Associative Property of Addition:

If a, b, and c are any numbers, then:

a + (b + c) = (a + b) + c

Example: Show that (2 + 3) + 4 = 2 + (3 + 4).

(P2) Commutative Property of Multiplication:

If a and b are any numbers, then:

a × b = b × a

Example: Verify that 2 × 3 = 3 × 2.

Lecture 2: Distributive Property

(P3) Distributive Property:

If a, b, and c are any numbers, then:

a × (b + c) = a × b + a × c

Example: Use distributive property to simplify 2 × (3 + 4).
