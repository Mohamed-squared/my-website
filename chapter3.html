Lecture 6 (Chapter 3)

Here we will start to study functions. In the words of Michael Spiral:

Undoubtedly, the most important concept in all of mathematics is that of a function - in almost every branch of modern mathematics, functions turn out to be the central object of investigation.

A function is an association of a value to every value in the function's "domain".

Lecture 7

Here we will look at examples of functions to get a deeper understanding of how functions work.

Example: A function that assigns each human (H) to its mother (M)

Function assigning each human to their mother

We know from this example that functions assign one output for every defined input.

Inputs can't have more than one output, but outputs can have more than one input.

Example: A function that outputs the age for every human.

p(p) = { age if p is male; age/2 if p is female }

We know from this example that functions can have more than one condition.

Real numbers can't sufficiently deal with infinities. Functions solve this problem.

Lecture 8

We represent the function mathematically like:

"name of the function": "domain" → "codomain"

"argument" → "value" (rule)

Example: A function with the name f and domain ℕ and the rule that relates the argument x to the value y = 4 - x has the notation:

f: ℕ → 4 - x

A function with an undefined domain and codomain is incomplete.

Another famous notation of functions is f(x) which is read as:

The value of function at argument x.

Note: If the domain is not specified, you assume it's all of the real numbers, as long as the output is defined.

Lecture 9

Example: f(x) = 8 + 3.6 · 9.7 · x

f(0) = 8    f(10) = 357.2    f(20) = 706.4
f(30) = 1055.6    f(40) = 1404.8    f(120) = 4198.4

This function represents motion under constant acceleration, specifically Free Fall.

Lecture 10

Combining functions can be done in multiple ways. For example, assume

x → f → f(x) = x²

x → g → g(x) = 6/(x-2) - 1

We can combine these two functions like

x → (f+g) → (f+g)(x) = f(x) + g(x)

x → (f-g) → (f-g)(x) = f(x) - g(x)

x → (f·g) → (f·g)(x) = f(x) · g(x)

x → (f/g) → (f/g)(x) = f(x) / g(x)

Note: The domain of each combined function is the domain of f ∩ the domain of g.

And the division function has an additional condition such that:

{x: g(x) ≠ 0}

We can also combine functions into composite functions:

x → (f∘g) → (f∘g)(x) = f(g(x))

Lecture 11

In the same way, we combined functions, we can decompose them in terms of simpler functions.

The simplest function is the identity function:

x → I → I(x) = x

Example: Decompose this function f(x) = (x + x² + sin(x) - sin²(x)) / (3x + x·sin(x) + sin(x²)):

f = (I + I·I + sin - sin·sin) / (I + I + I + I·sin + sin·(I·I))

Here we represented functions in terms of functions, not in terms of its outputs, making this notation mathematically more accurate.

Another way to represent the function notation is:

f: ℝ → ℝ

x → (x + x² + sin(x) - sin²(x)) / (3x + x·sin(x) + sin(x²))

This shows that there are three ways to represent a function:

  1. As a least accurate representation of the value.
  2. As the most accurate representation in the language of functions.
  3. As an intermediate notation.

Lecture 12

Our first definition of functions was:

"A function is a rule which assigns, to each of certain real numbers, some other real number."

This definition is ambiguous because in this definition if the rule is different, even if the result is the same, the function is different, which is wrong.

The key to finding the definition is to know that a function is a process of pairing two numbers.

So we can define functions to be:

"A function is a collection of pairs of numbers with the following property: if (a, b) and (a, c) are both in the collection, then b = c; in other words, the collection must not contain two different pairs with the same first element."

Another definition is:

"If f is a function, the domain of f is the set of all a for which there's some b such that (a, b) is in it. If a is in the domain of f, it follows from the definition of a function that there's, in fact, a unique number b such that (a, b) is in f. This unique b is denoted by f(a)."

These are the official definitions of functions.

But nothing can hold you from thinking about functions with the old definition. However, you won't be able to know what a function really is without graphing it.
