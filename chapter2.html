Lecture 5 (Chapter 2)

The 12 properties of numbers that we listed cannot fully describe how all numbers work. The types of numbers are:

  1. Natural numbers (counting numbers) $\mathbb{N} = \{1, 2, 3, \ldots\}$. All properties apply except P2, P3, P7.

    The most basic property of $\mathbb{N}$ is the principle of mathematical induction. We can prove that the property $P$ is applied for all numbers if:

    1. $P(1)$ is true.
    2. Whenever $P(k)$ is true, $P(k + 1)$ is true.
    Example: Prove that the sum of $1 + 2 + \ldots + n = \frac{n(n + 1)}{2}$ 
    1 + \cdots + k = \frac{k(k + 1)}{2} \rightarrow 
1 + \cdots + k + (k + 1) = \frac{k(k + 1)}{2} + (k + 1)
                         = \frac{k(k + 1) + 2(k + 1)}{2}
                         = \frac{k^2 + 3k + 2}{2}
                         = \frac{(k + 1)(k + 2)}{2}

    And there's another form of induction called complete induction:

    1. $1$ is in $A$.
    2. If $1, \ldots, k$ is in $A$, then $k + 1$ is in $A$.

    A closely related topic to mathematical induction is recursive definitions. For example, the number $n!$ is defined as:

    1. $1! = 1$
    2. $n! = n \cdot (n - 1)!$
  2. Whole numbers $\mathbb{W}$ (includes $0$). All properties apply except P3, P7.
  3. Integers $\mathbb{Z}$ (includes negative numbers). All properties apply except P7.
  4. Rational numbers $\mathbb{Q}$ (can be expressed as fractions). All properties apply.
  5. Irrational numbers (cannot be expressed as fractions, e.g., $\pi$, $e$, $\sqrt{2}$). All properties apply.
    Proof: That $\sqrt{2}$ is irrational (suppose $\sqrt{2} = \frac{p}{q}$) 
    \frac{p^2}{q^2} = 2 \rightarrow p^2 = 2q^2
(Assume $p$ is even, let $p = 2k$)
4k^2 = 2q^2 \rightarrow 2k^2 = q^2
(This implies $q$ is even)

    So, $\frac{p}{q}$ is not in simplest form, contradicting our assumption.

    But we can't get the value of any irrational number or prove their existence using our ordinary number properties.
